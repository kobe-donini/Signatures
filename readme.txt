Instruções: 

Coloque o nome do colaborador no arquivo no seguinte formato

nome,função,url_da_imagem

Exemplo:

Rodrigo Freitas Leite,CTO,https:\/\/kobe.io\/wp-content\/uploads\/2020\/02\/Rodrigo.jpg

Não se esqueça que de colocar uma contra barra se tiver uma barra no endereço da imagem. 


No console a direita utilizar o comando 

bash main.sh people.txt 

Para fazer o download basta entrar no link do respositório: https://github.com/kobe-donini/Signatures e baixar o conteúdo da pasta
